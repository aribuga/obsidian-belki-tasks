# Sorting and filtering

## Sorting

The **Sorting** menu is available in every view. Click the **Sorting** button in the top-right area of the task board.

Available sort modes:

| Mode | Description |
|---|---|
| Smart | Balances due date, priority, and creation order |
| Due date | Earliest due date first |
| Priority | Highest priority (P1) first |
| Deadline | Earliest deadline first |
| Created date | Newest tasks first |
| Project | Alphabetical by project name |
| Alphabetical | A–Z by task title |

The selected sort mode is saved and persists between sessions.

---

## Grouping (project views only)

Inside a project view, you can group tasks into sections using the **Group by** option at the bottom of the Sorting menu.

| Group by | Sections |
|---|---|
| None | Flat list (default) |
| Label | One section per label; tasks with no label go under "No label" |
| Priority | P1, P2, P3, P4, No priority |

- Empty groups are not shown.
- Sorting still applies within each group.
- Tasks with multiple labels appear under their first label only.
- Group by is only available in project views, not in Today, Upcoming, or other views.

## Filtering

### Filters & Labels view

The **Filters & Labels** view shows pre-built filters:

- By priority (P1–P4)
- View all active tasks
- No due date
- Due today
- Overdue
- Has a deadline
- No label

Click a filter row to see the matching tasks. Sorting applies inside filtered results.

### Label filtering

Click any label chip on a task card to jump to the filtered view for that label. You can also click a label row in the **Filters & Labels** view.

---

## Today view grouping

The Today view always shows two fixed groups:

- **Today** — tasks due today
- **Overdue** — past-due tasks, within the configured overdue range

The overdue range can be set in the header of the overdue section:

- Yesterday
- Last 7 days
- Last 30 days
- Older

---

## Upcoming view grouping

The Upcoming view groups tasks by due date into labeled date sections (e.g., "Mon 30 Jun", "Tue 1 Jul"). Sorting applies within each date group.

---

## Completed view grouping

The Completed view groups tasks by their completion date:

- Today
- Yesterday
- Specific date
- Earlier (tasks with no recorded completion date)
