# Manual regression checklist

## Context-aware Quick Add shortcut

1. Open a regular Markdown note.
2. Press `Cmd/Ctrl + Shift + A`.
3. Confirm the global Quick Add modal opens.
4. Create a task and confirm it is added to Inbox.
5. Open Belki's Today view.
6. Press `Cmd/Ctrl + Shift + A`.
7. Confirm the desktop floating composer opens.
8. Confirm Today is applied as the contextual due date.
9. Open a project view and use the shortcut.
10. Confirm the current project is selected.
11. Open Upcoming and use the shortcut.
12. Confirm Upcoming's existing contextual defaults are preserved.
13. Open the composer and use the shortcut again.
14. Confirm no duplicate composer appears and the existing title input receives focus.
15. Keep Belki open in a background tab, activate a Markdown note, and use the shortcut.
16. Confirm the global Quick Add modal opens rather than the Belki composer.
17. Test the command on mobile.
18. Confirm the desktop floating composer does not appear and existing mobile behavior is unchanged.
19. Assign a custom shortcut through Obsidian settings.
20. Confirm the custom shortcut continues to override the default.
